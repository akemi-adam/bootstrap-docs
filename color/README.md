# Cor

A parte referente a cores no Bootstrap é bem fácil, podendo ser abordada apenas nesse documento mesmo.

Temos as classes text-color, que são elas:

```
text-muted
text-success
text-info
text-warning
text-danger
text-secondary
text-white
text-dark
text-body
text-light
```

Para as classes de background-color, temos:

```
bg-primary
bg-success
bg-info
bg-warning
bg-danger
bg-secondary
bg-dark
bg-light
```
